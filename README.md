# ⚡ FastAPI e Python: APIs com Autenticação, Banco de Dados e Deploy

Projeto desenvolvido durante o curso **"FastAPI e Python: construindo APIs com autenticação, banco de dados e deploy"** da [Alura](https://www.alura.com.br), como parte da trilha de **Desenvolvimento Backend com Python**.

---

## 📚 Sobre o Projeto

API completa construída com **FastAPI**, cobrindo desde a estruturação de rotas e modelos até autenticação, testes automatizados e deploy com Docker.

---

## ✅ O que foi implementado

- Estruturação de rotas e modelos com **Pydantic** e documentação automática via **Swagger**
- Operações **CRUD** com persistência em **SQLite**, organização em camadas
- Programação assíncrona com **async/await** e controle do event loop
- Autenticação com **cookies HTTP**, controle de sessão e **middlewares** de proteção de rotas
- Integração front-end e back-end com **Jinja** e templates HTML
- Testes automatizados com **PyTest**, fixtures, `MagicMock` e `AsyncMock`
- Deploy com **Docker** e estrutura modular para produção

---

## 🛠️ Tecnologias Utilizadas

- **Python 3.x**
- **FastAPI**
- **Pydantic**
- **SQLite**
- **Jinja2**
- **PyTest**
- **Docker**
